# Сайт «СОПОДЧИНЕНИЕ» (HTML + CSS + JS)

Сделано по вашим слайдам: титул + введение (объединены на главной), схема с одной Землёй и уровни таксономии.

## Файлы
- `index.html` — Главная (титул + введение)
- `scheme.html` — Схема таксономии (одна Земля + полный список уровней)
- `example.html`, `team.html`, `reviews.html` — пока заглушки
- `css/style.css` — стили
- `js/app.js` — подсветка активной кнопки меню
- `assets/bg-earth.jpg` — фон-земля
- `assets/earth-circle.png` — круглая Земля для схемы

## Запуск
Откройте `index.html` в браузере (двойной клик).
